# CommandLineHere
Small AutoHotKey Script to create "open here" keybinds for tools i use

Current Keybindings:

| keybind             | effect             | comments            |
| ------------------- | ------------------ | ------------------- |
| Windows + C         | Open cmd           |                     |
| Windows + Shift + C | Open vscode        |                     |
| Ctrl + Shift + M    | Open Sublime Merge |                     |
| Windows + Shift + R | Open Toolman       | a custom tool I use |
